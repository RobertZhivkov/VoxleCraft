# VoxleCraft
VoxleCraft е игра подобна на Minecraft в която се строй с voxLE-и

Бележка:
  -За по нови верси от alpha-1.1.2, моля изтеглете 'assets' папката и го сложете в същата папка до verstion-x.x.x.py

Платформи:
  Windows 8.1
  windows 10,
  Windows 11,
  Linux (Всеки)

Задължитолно истегляне:
  - Python (3.12+): https://www.python.org/
  - Ursina: https://www.ursinaengine.org/index.html/

След теглене, пуснете приложението да играете
